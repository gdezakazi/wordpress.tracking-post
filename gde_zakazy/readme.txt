1. Предназначение сервиса ГДЕЗАКАЗЫ.РФ
• Облачный сервис отслеживания посылок Почты России для интернет магазинов создан для увеличения прибыли за счет:
• Снижения количество возвратов посылок
• Снижения риск кражи посылки в отделениях Почты России
• Снижения нагрузки на оператора телефонных звонков
• Повышения качество обслуживания Ваших клиентов
• Получения обратной связи от Ваших клиентов

Сервис ГДЕЗАКАЗЫ.РФ имеет ряд преимуществ над аналогами:
• Отслеживание всегда бесплатно через веб-интерфейс
• Отправка email всегда бесплатна
• Стоимость отправки смс высчитывается для оператора индивидуально
• Нет подписок на отправку смс


2. Функциональные возможности модуля
Данный модуль полностью автоматизирует процесс получения посылки клиентом за счет:
• Автоматического отслеживания трекингов Почты России
• Уведомления по смс, по почте вашим клиентам о прохождении посылки
• Автоматической смене статусов заказа

После занесения номера трекинга посылки во вкладке к заказу, информация передается на сервер ГДЕЗАКАЗЫ.РФ, после чего в течение дня по несколько раз происходит проверка статуса данной посылки. Посылки сгруппированы несколькими статусами: "Новые", В пути", "В отделении", "Проблемные", "Доставлены", "Архив".

Для каждого из перечисленных статусов в личном кабинете ГДЕЗАКАЗЫ.РФ, менеджер интернет-магазина настраивает необходимые смс и email уведомления своим клиентам. Вы можете менять заготовленные шаблоны на свои тексты. Для смс существует возможность отправки сообщения на кириллице или латинице, от своего буквенного отправителя в соответствии с названием магазина. Также необходимые сообщения приходят менеджеру интернет-магазина.

Модуль автоматически меняет статус заказа в соответствии со сменой статуса нахождения посылки. При этом, вы можете настроить готовое сообщение при статусе заказа использовав переменную с номером трекинга.

Например, для получения обратной связи от клиента в статусе "Заказ доставлен" вы с легкостью можете попросить оставить отзыв о вашем магазине на сайтах-отзовиках или в социальных сетях. Предусмотрена ситуация с неудачным заказом, когда посылка оказалось "Проблемной" - долго доставлялась, отправилась не по тому адресу, долго не забиралась. В таком случае отзыв запрашиваться не будет.

3. Установка
1. В панели администратора Wordpress выберите пункт меню Плагины - Добавить новый
2. Загрузите файл gde_zakazy.zip
3. Нажмите на кнопку Загрузить плагин и выберите ранее загруженный файл плагина
4. На странице Плагины - Установленные плагины найдите модуль GDEZAKAZI.RU и нажмите кнопку активации модуля
5. После установки зайдите в настройки модуля и настройте модуль
6. Зарегистрируйтесь на сайте ГДЕЗАКАЗЫ.РФ и ОБЯЗАТЕЛЬНО подтвердите ваш email.
7. Перейдите в раздел Настройки - Настройки API и скопируйте ключ API.
8. Вставьте ключ API в настройках модуля вашего OpenCart.
